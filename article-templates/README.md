# SleepLemon Article Templates

Copy one template into `src/content/posts/`, rename the file with a URL-friendly slug, then edit the frontmatter and article body.

Examples:

```bash
cp article-templates/zh-sleep-article-template.md src/content/posts/zh-evening-light-routine.md
cp article-templates/zh-affiliate-guide-template.md src/content/posts/zh-white-noise-guide.md
```

Use `affiliate: false` for normal sleep content.

Use `affiliate: true` only when the article contains shopping links. The site will show the Amazon Associate disclosure on those pages.

Keep only one or a few important posts as `featured: true`, because featured posts appear on the homepage.
